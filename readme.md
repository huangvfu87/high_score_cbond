## 📊 核心识别目标

**我们要找的是什么？**
- 不是暴涨暴跌的妖债
- 而是**稳健向上、持续上涨**的可转债

## 🎯 评分体系（100分制）

### 1️⃣ 连续性指标（30分）- 最核心

**目的**：判断上涨是否持续、稳定

**指标说明**：
- **5日上涨天数**：最近5天有几天收盘价高于前一日
  - 5天中4-5天上涨 → 得15分（非常强）
  - 5天中3天上涨 → 得10分（较强）
  
- **10日上涨天数**：最近10天有几天上涨
  - 10天中7-10天上涨 → 得15分
  - 10天中6天上涨 → 得10分

**判断逻辑**：
```
如果一只转债：
- 5天中4天上涨，1天小跌
- 10天中8天上涨，2天小跌
→ 这就是典型的"连续小阳"形态，得分高
```

---

### 2️⃣ 均线系统（25分）- 趋势判断

**目的**：判断是否处于上升趋势通道

**核心指标**：
- **均线多头排列**（10分）
  - MA5 > MA10 > MA20 > MA30
  - 说明短期、中期、长期都在上涨
  
- **价格与5日均线关系**（8分）
  - 价格在5日均线上方0-5%之间最佳
  - 太贴近：力度不够
  - 太远：可能回调
  
- **站稳5日线天数**（7分）
  - 近10天中有7天以上收盘价>5日均线
  - 说明均线支撑有效

**判断逻辑**：
```
你图片中的走势：
- K线始终在均线上方运行
- 均线呈多头排列（短期>长期）
- 偶尔回踩均线但不跌破
→ 这就是标准的均线多头形态
```

**形象理解**：
- 均线就像**楼梯**
- 价格沿着楼梯拾级而上
- 偶尔停顿但不往下走

---

### 3️⃣ 上涨质量（20分）- 风险控制

**目的**：判断上涨是否健康、可持续

**核心指标**：
- **5日平均涨幅**（10分）
  - 理想范围：日均0.5% - 3%
  - 太慢：没机会
  - 太快：不稳定、容易回调
  
- **最大单日回撤**（10分）
  - 5天内最大跌幅 < 3%
  - 说明没有大幅回调，上涨稳健

**判断逻辑**：
```
好的走势：
日1: +1.5%
日2: +2.0%
日3: +0.8%
日4: -1.0%  ← 小回调
日5: +1.8%
→ 平均每天涨1%，最大跌1%，很稳健

坏的走势：
日1: +8%
日2: -5%
日3: +6%
日4: -4%
→ 虽然也在涨，但波动太大，风险高
```

---

### 4️⃣ 低点抬高（10分）- 底部形态

**目的**：判断是否真正的上升趋势

**指标说明**：
- 比较最近5天的**最低价**
- 如果第5天的最低价 > 第1天的最低价
- 说明即使回调，底部也在不断抬高

**形象理解**：
```
价格走势：
  ╱╲  ╱╲  ╱╲
 ╱  ╲╱  ╲╱  ╲
←低1  低2  低3→

低1 < 低2 < 低3  ← 底部抬高，这是上升趋势
低1 = 低2 = 低3  ← 横盘震荡
低1 > 低2 > 低3  ← 底部下移，这是下降趋势
```

---

### 5️⃣ 动量指标（15分）- 买卖时机

**目的**：判断当前位置是否适合买入

**核心指标**：

**RSI - 相对强弱指标**（8分）
- 理想区间：50-70
- < 30：超卖（可能反弹，但我们不要）
- 50-70：强势但未超买（最佳）
- \> 70：超买（警惕回调）

**MACD - 趋势指标**（7分）
- DIF > 0 且向上 → 多头趋势
- DIF < 0 → 空头趋势

**判断逻辑**：
```
RSI = 45 → 刚开始走强，可以买
RSI = 65 → 强势但未过热，好
RSI = 85 → 过热，可能要回调，不买
```

## 💡 核心识别逻辑总结

**用大白话说就是**：

1. **连续上涨**：不是偶尔涨一次，而是"连续"涨多次
2. **涨得稳健**：不是暴涨暴跌，而是每天涨一点
3. **有均线支撑**：价格站在均线上方，有支撑
4. **底部抬高**：每次回调的低点越来越高
5. **未过度炒作**：RSI不要太高，避免追高

**形象比喻**：
- ❌ 不要：坐火箭（上去快，下来也快）
- ✅ 要找：爬楼梯（虽然慢但稳，持续向上）

**你图片中的案例就是典型的"爬楼梯"型**：
- 连续小阳线
- 沿着均线稳步上升
- 偶尔小回调但不破坏趋势
- 最终形成持续上涨
